# Meta Content Library: Academic Access Guide

## What is Meta Content Library?

Meta Content Library (MCL) is the official replacement for CrowdTangle, providing academic researchers access to public Facebook and Instagram content via a Python API and a secure Researcher Platform.

## How to Apply

1. Visit [Meta Content Library](https://transparency.fb.com/researchtools/meta-content-library/)
2. Click **Apply for Access**
3. Complete the academic researcher application (institutional affiliation required)
4. Await approval (typically 2–6 weeks)
5. Once approved, access the Researcher Platform at [researchers.facebook.com](https://researchers.facebook.com)

## Access Tiers

| Tier | Access Level |
|---|---|
| UI-only | Browser-based search and export |
| API access | Python API with programmatic data collection |
| Enhanced | Larger data volumes for approved studies |
