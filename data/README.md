# Data Notes

Raw data files are excluded from this repository via `.gitignore`.

Store your collected data here as:
- `data/raw/crowdtangle_export.csv` (CrowdTangle legacy)
- `data/raw/mcl_facebook_posts.csv` (Meta Content Library)
- `data/raw/mcl_instagram_posts.csv`

All raw files must remain local and must not be committed to GitHub.
